-- Proper Openfire artefacts in maven --

For building the plugins, a openfire artecfact is needed. As the correct openfire jar is a collection of all releveant dependencies, this is not really usefull as a maven artefact.

The directory contains a pom file, which refers to openfire jar which only contains the openfire classes as found in the src tree ofopenfire, without the classes extracted from the dependencies.

This jar is generated by adjusting the openfire build o create and bare-openfire jar.
